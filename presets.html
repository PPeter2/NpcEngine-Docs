Presets & Archetypes

Using Presets
Apply a preset

The fastest way to create an NPC. Presets:Apply() converts a preset into a config table ready for Engine.new().

local Presets = NPCEngine.Presets

local npc = NPCEngine.new(
    Presets:Apply(Presets:Get("Guard"), workspace.MyNPC)
)

-- Apply the preset's perception config too
local preset = Presets:Get("Guard")
if preset.perception then
    npc:ConfigurePerception(preset.perception)
end
Extend a preset

Create a custom preset that inherits from a built-in archetype. Only override the fields you want to change — everything else is inherited.

Presets:Register("EliteGuard", Presets:Extend("Guard", {
    MoveSpeed  = 22,
    SleepRange = 200,
    perception = {
        sightRange = 90,
        sightFOV   = 130,
        memoryTime = 12,
    },
}))

-- Now use it like any other preset
local npc = NPCEngine.new(
    Presets:Apply(Presets:Get("EliteGuard"), workspace.EliteNPC)
)
Hot-swap at runtime

Change an NPC's preset while it is live — updates move speed, sleep range, and perception without rebuilding the instance.

-- Boss enrages at 50% health — swap to a faster preset
humanoid.HealthChanged:Connect(function(hp)
    local pct = hp / humanoid.MaxHealth
    if pct < 0.5 then
        Presets:HotSwap(npc, "Boss")
        print("Boss enraged!")
    end
end)
List all presets
local names = Presets:List()
